# robot_servo_control
A very simple flutter control.

Bluetooth Control pan/tilt servo motor 
Este aplicativo foi desenvolvido em Flutter 3.0 
Principais Bibliotecas do app:
flutter_bluetooth_serial

Principais Bibliotecas do esp32:

BluetoothSerial.h
